## Run Locally

Clone the project

```bash
  git clone https://github.com/enricoirawan/test-project
```

Go to the project directory

```bash
  cd https://github.com/enricoirawan/test-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
